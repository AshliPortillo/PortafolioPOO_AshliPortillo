**Proyecto 5- Actividad de evaluación - Gestión de Bitácoras en Archivos de Texto (PHP)**

- Objetivo del proyecto

Desarrollar un sistema básico en PHP para registrar actividades de una bitácora, almacenarlas en un archivo de texto y mostrarlas en pantalla, aplicando manejo de archivos y validación de datos.

- Problema que resuelve

Resuelve la necesidad de llevar un control digital sencillo de actividades diarias de una empresa sin necesidad de usar una base de datos, permitiendo registrar, guardar y consultar información de manera organizada.

- Tecnologías utilizadas
PHP
HTML
XAMPP (Apache)
Archivo de texto (.txt)

- Conceptos aplicados (según temario)
Formularios HTML
Manejo de archivos en PHP
file_put_contents()
file()
FILE_APPEND
Validación de datos
Condicionales
Lectura y escritura de archivos
Uso de htmlspecialchars()

- Instrucciones de ejecución
Copiar la carpeta bitacora dentro de htdocs
Iniciar Apache en XAMPP
Abrir el navegador
Escribir: localhost/bitacora
Llenar el formulario
Guardar la actividad
Verificar que los registros aparezcan en pantalla y se almacenen en bitacora.txt

- Reflexión personal

Con este proyecto reforcé el uso de archivos en PHP y comprendí cómo almacenar y recuperar información sin usar bases de datos. Aprendí la importancia de validar datos y organizar mejor la información, además de que este conocimiento puede servir para desarrollar sistemas sencillos de registro y control.
