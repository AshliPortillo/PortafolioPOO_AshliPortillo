**Proyecto 4 - Actividad Integradora - Arreglos unidimensionales**

- Desarrollar un sistema básico en PHP que permita registrar productos y precios, almacenarlos en arreglos unidimensionales y procesarlos para obtener información importante del inventario, como el total, promedio, producto más caro y producto más barato.

- Problema que resuelve
Resuelve la necesidad de una tienda en línea de organizar y analizar su inventario de manera sencilla. Permite capturar datos de productos y obtener automáticamente resultados útiles para el control de precios.

- Tecnologías utilizadas

PHP
HTML
XAMPP
Servidor Apache
Navegador web

- Conceptos aplicados (según temario)

Arreglos unidimensionales
Arreglos paralelos

Validación básica de datos
Ciclos for
Procesamiento de formularios con PHP
Funciones predefinidas de PHP (array_sum(), max(), min(), array_search(), count())
Generación dinámica de tablas HTML

- Instrucciones de ejecución

Copiar la carpeta del proyecto dentro de la carpeta htdocs de XAMPP.
Abrir el panel de control de XAMPP.
Iniciar el servicio Apache.
Abrir el navegador web.
Escribir en la barra de direcciones:
localhost/nombre_del_proyecto
Capturar los nombres y precios de los productos.
Presionar el botón de procesar para visualizar los resultados.

- Reflexión personal

En este proyecto aprendí a usar arreglos unidimensionales en PHP para almacenar y procesar datos, además de aplicar funciones que facilitan cálculos como sumas, máximos y mínimos. Lo más difícil fue relacionar correctamente los productos con sus precios al momento de procesarlos. Considero que podría mejorar el diseño del programa y agregar más opciones para hacerlo más dinámico y funcional.
